COMPONENTES DE LA PC

CPU

El CPU (Unidad Central de Procesamiento) es el cerebro de la computadora, encargado de ejecutar las instrucciones de los programas, dictando tareas a todos los demás componentes.

GPU

Las tarjetas gráficas independientes, son componentes grandes y potentes que se conectan a la ranura PCIe x16 de la placa base de la PC.

Motherboard

La placa base es la placa de circuitos principal y está conectada a todo. La CPU se encuentra directamente en la placa base y todos los demás componentes (tarjetas gráficas, discos duros, memoria, unidades ópticas, tarjetas inalámbricas) se integran en la placa base.

RAM

La memoria de acceso aleatorio (RAM) es la memoria a corto plazo de tu PC. Es donde las aplicaciones almacenan datos temporales a los que se debe acceder rápidamente; son esas “listas de instrucciones” que lee y ejecuta la CPU.

Disco duro (HDD).

Los discos duros, también conocidos como HDD, son un componente informático que sirve para almacenar de forma permanente tus datos. Esto quiere decir, que los datos no se borran cuando se apaga la unidad como pasa en los almacenados por la memoria RAM. Están compuestos de piezas mecánicas, de ahí que a veces se le llame discos duros mecánicos, y utilizan el magnetismo para grabar tus datos y archivos.

unidades de estado sólido (SSD)

Las unidades de estado sólido o SSD (Solid State Drive) son una alternativa a los discos duros. La gran diferencia es que mientras los discos duros utilizan componentes mecánicos que se mueven, las SSD almacenan los archivos en microchips con memorias flash interconectadas entre sí. Por lo tanto, casi podríamos considerarlos como una evolución de las memorias USB.

Fuente de alimentación (PSU)

Las unidades de estado sólido o SSD (Solid State Drive) son una alternativa a los discos duros. Refrigeración

hay diversos tipos de refrigeración. Todo depende de las características del equipo y del uso que quieras darle a tu torre. La refrigeración pasiva se consigue a través de una parte metálica con aletas de aluminio o cobre que dispersan el calor de un modo simple, económico y duradero. La refrigeración activa, que suma ventiladores de apoyo o un disipador líquido con ventilación.

Perifericos

un periférico es un dispositivo independiente, conectado externa o internamente a la tarjeta madre, que permite al sistema informático realizar una función extra. Como tal, no forma parte del proceso central de una computadora, sino que sirve a nivel de complemento para aumentar sus capacidades funcionales. Los dispositivos periféricos son los responsables de realizar las operaciones de entrada y salida de un sistema informático

Sistema Operativo (SO)

El sistema operativo es el software que coordina y dirige todos los servicios y aplicaciones que utiliza el usuario en una computadora, por eso es el más importante y fundamental. Se trata de programas que permiten y regulan los aspectos más básicos del sistema.

BIOS

La BIOS pone las funciones básicas de un ordenador a disposición del usuario. Al encenderlo, comprueba si las partes más importantes son funcionales, como la memoria, la CPU y otros componentes de hardware.
